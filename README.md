# Cloud Infrastructure Optimization using Unsupervised Learning

## 📌 Overview
This project applies unsupervised machine learning techniques to analyze and optimize cloud infrastructure configurations.

## 🚀 Objective
To group similar system configurations based on features like CPU, memory, storage, bandwidth, and energy consumption for better resource allocation.

## ⚙️ Features
- Data Cleaning & Preprocessing
- Missing Value Handling
- Feature Scaling
- K-Means Clustering
- PCA / t-SNE Visualization

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Output
Clusters of similar infrastructure setups to support efficient decision-making in cloud environments.
